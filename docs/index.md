# Blog Editor 0.7

## 更新日志

- 用户可以自由登陆，登出编辑器
- 用户可以选择博文进行编辑，也可以创建新的博文。
- 支持拖拽照片并上传至互联网
- 支持发布新的博文
- 现在用户只能对自己的博文进行编辑，除非是超级用户。

## 简介

博客编辑器是一个在线博客编辑器。它使用 Electron，React 技术，实现直观式博客正文编辑功能。 用户可以对正文自由编辑，排版，增加格式，而无需掌握任何编程技术。
